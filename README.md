# Описание проекта

Telegram Restaurant Service Bot — это учебный проект, разработанный в рамках курса по объектно-ориентированному программированию. Бот предоставляет базовые функции взаимодействия с клиентами ресторана через Telegram.

## Основные возможности

Регистрация пользователей по команде /start

Хранение данных пользователей:
 + ID чата
 + Имя и фамилия
 + Имя пользователя (username)
 + Время регистрации
 + Персонализированные ответы на сообщения

Простое расширение функционала

## Технологический стек

Язык программирования: Java 17
Фреймворк: Spring Boot 3.1
База данных: PostgreSQL / H2 (для разработки)
Telegram API: telegrambots-spring-boot-starter 6.9.7
Дополнительно: Lombok, Maven

## Быстрый старт

Необходимые компоненты: 
JDK 17
Maven 3.8+
Токен бота Telegram (получить у @BotFather)
PostgreSQL (опционально)
